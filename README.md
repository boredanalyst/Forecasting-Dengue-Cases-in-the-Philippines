# Forecasting-Dengue-Cases-in-the-Philippines

This project analyzes dengue case trends in the Philippines, focusing on the National Capital Region, by applying Holt‑Winters exponential smoothing to forecast post‑pandemic dynamics. Using historical datasets from 2016 onward, the model captures both long‑term trends and seasonal cycles, such as rainy‑season spikes and dry‑season dips. The forecasts highlight how pandemic disruptions caused sharp declines in reported cases, while multiplicative seasonality ensures realistic projections that remain positive and scale with the baseline. By combining statistical rigor with epidemiological context, the project provides transparent, interpretable forecasts that can support public health planning and demonstrate the value of time‑series methods in real‑world applications.
